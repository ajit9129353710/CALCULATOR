🧮 Simple Calculator

A simple calculator built using HTML, CSS, and JavaScript.

This project can perform basic arithmetic operations like addition, subtraction, multiplication, and division.

✨ Features

- ⭐ Addition
- ⭐ Subtraction
- ⭐ Multiplication
- ⭐ Division
- ⭐ Decimal numbers
- ⭐ Clear button
- ⭐ Delete last character
- ⭐ Simple and responsive UI

🛠️ Technologies Used

- HTML
- CSS
- JavaScript

📂 Project Files

- 📄 "index.html"
- 🎨 "style.css"
- ⚡ "script.js"

🚀 How to Use

1. Open "index.html" in your browser.
2. Click the buttons to enter numbers and operators.
3. Press = to get the result.
4. Use C to clear the display.
5. Use DEL to remove the last entered character.
